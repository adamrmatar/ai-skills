# Claude Co-work Workflows

Use this skill to optimize your interaction model when pair programming or co-working with Claude. Address Claude as an active engineering collaborator and leverage Projects and Artifacts.

## 🛠️ Step-by-Step Co-working Workflow

1. **Set Up a Project Workspace**: Initialize a Claude Project and upload context documents (e.g., API schemas, architectural diagrams, UI styling guidelines).
2. **Configure Custom Instructions**: Set Project Instructions to define coding styles (e.g., "Use TypeScript with strict typing, follow DRY, and keep files under 200 lines").
3. **Use Artifacts for Deliverables**: Let Claude render large code blocks, HTML mockups, or documents into the Artifacts panel to keep chat logs clean and focused.
4. **Iterate Incrementally**: Address changes by pointing to specific sections of the active artifact. Refine step-by-step rather than asking for full rewrites.
5. **Session Housekeeping**: Start a new chat thread once a specific task is finished to keep the context window small and prevent response lag.
