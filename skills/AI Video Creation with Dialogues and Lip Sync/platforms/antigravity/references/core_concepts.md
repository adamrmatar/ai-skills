# Core Concepts
Creating AI videos involves several core concepts that streamline the process and ensure high-quality output. Understanding these concepts is crucial for effective video creation.

## Story Generation
The first step in creating an AI video is generating a compelling story. This can be done by inputting an idea, providing a YouTube link, or selecting a theme. The magic tool automates this process, creating a complete script based on your input.

## Scene Division
Once the story is generated, it is divided into scenes. Each scene represents a part of the story and is essential for creating a coherent video. The tool provides options to review and adjust the scene division.

## Image Creation
Images for each scene are created using Chat GPT. This involves generating visuals that match the scene's description. The images are then used as the basis for video clips.

## Video Conversion
The final step is converting the images into video clips using Flow AI. This involves uploading the images and using prompts to generate video clips. The clips are then assembled to create the final video.

Understanding these core concepts will help you navigate the video creation process efficiently and produce high-quality AI videos.
