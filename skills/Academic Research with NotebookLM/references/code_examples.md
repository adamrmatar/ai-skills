# Code Examples for Academic Research with NotebookLM

Here are some practical examples of prompts and commands you can use in NotebookLM to enhance your academic research.

## Literature Review Prompt
```markdown
What research methodologies are used across these studies? Compare their sample sizes, populations, and key limitations.
```

## Self-Critique Prompt
```markdown
What arguments am I making in my draft? Are they well supported by the literature sources?
```

## Data Table Prompt
```markdown
Create a data table summarizing the major findings in solar cell technology evolution.
```

## Mind Map Creation
```markdown
Create a mind map visualizing the themes and knowledge gaps in my sources.
```

## Visual Aid Creation
```markdown
Generate a slide deck summarizing the key findings from these papers.
```

These examples demonstrate how to use NotebookLM to extract and organize information effectively for your academic research.
