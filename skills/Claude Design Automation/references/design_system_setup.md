## Design System Setup Guide

1. Gather all brand assets (logos in multiple formats, color codes, font files)
2. Collect examples of existing designs (website screenshots, slide decks, social media posts)
3. Use the Design System Creator skill to:
   - Scrape your website for design elements
   - Organize assets into a structured folder
   - Generate Claude Design system prompts
4. Upload the complete folder to Claude Design
5. Test the system by generating sample assets
