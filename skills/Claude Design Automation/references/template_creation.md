## Template Creation Process

1. Identify repetitive design use cases (slide decks, social posts, etc.)
2. Collect layout inspirations from:
   - Gamma (for slides)
   - Dribbble/Behance (for UI/UX)
   - Framer (for websites)
3. Feed examples to Claude Design with instruction:
   "Recreate these layouts in HTML using our brand design system. Use placeholders for content."
4. Refine through 2-3 iterations
5. Save as template via Share > Duplicate as Template
