# Ai Agent Management And Local Model Deployment

Learn how to design, manage, and deploy AI agents effectively, including setting up local AI models for privacy, cost, and latency-sensitive workflows.

## Setup & Installation

This skill is agent-agnostic and packaged for multiple developer agents. Find your target platform below:

### 💻 Antigravity
_Copy the contents of `platforms/antigravity/` to `~/.gemini/config/skills/AI Agent Management and Local Model Deployment/`._

### 💻 Cursor
_Copy `platforms/cursor/AI Agent Management and Local Model Deployment.mdc` to your project's `.cursor/rules/` directory._

### 💻 Claude
_Append the contents of `platforms/claude/CLAUDE.md` to the target project's `CLAUDE.md` file._

### 💻 Copilot
_Copy `platforms/copilot/AI Agent Management and Local Model Deployment.instructions.md` to your project's `.github/instructions/` directory._

### 💻 Windsurf
_Append the contents of `platforms/windsurf/.windsurfrules` to the target project's `.windsurfrules` file._
