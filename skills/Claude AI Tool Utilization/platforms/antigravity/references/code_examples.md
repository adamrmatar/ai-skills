# Code Examples for Claude AI

Here are some practical code snippets and prompts for using Claude AI:

1. **Trip Planning Prompt**:
   ```
   I'm traveling to Bengaluru on [date]. Suggest some best vegetarian restaurants which offer less spicy and tasty food. I prefer locations near HSR Layout.
   ```

2. **Document Analysis Prompt**:
   ```
   I live in NJ and this is the estimate I got for my car repair. Tell me if the garage is overcharging.
   ```

3. **Financial Dashboard Prompt**:
   ```
   Build me an HTML simulation for a scenario where the flat price is three crore and the interest rate is 8%. Compare buying vs. renting.
   ```

4. **File Organization Prompt**:
   ```
   Look at all my files in this folder, organize them into subfolders by expense type, rename each file with clear format date, vendor, amount.
   ```

5. **Chrome Extension PRD Prompt**:
   ```
   I want to build a Chrome extension that helps in capturing highlights on any website and storing them. Create a PRD in markdown format.
   ```

These examples show how to structure prompts for optimal results.
