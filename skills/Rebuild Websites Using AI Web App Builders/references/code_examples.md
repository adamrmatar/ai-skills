# Code Examples

## Prompt Templates

Here are some example prompts and responses to guide the AI through the website build process:

```plaintext
Prompt: What is the primary purpose of your website?
Response: To track and organize AI products.

Prompt: Which AI categories will your website cover?
Response: Machine Learning, Natural Language Processing, Computer Vision.
```

## Example Workflow

1. **Define Your Audience and Purpose**:
   ```plaintext
   Prompt: Who is your target audience?
   Response: AI enthusiasts and developers.
   ```

2. **Select AI Categories**:
   ```plaintext
   Prompt: Which AI categories will your website cover?
   Response: Machine Learning, Natural Language Processing, Computer Vision.
   ```

3. **Guide the AI Through the Build Process**:
   ```plaintext
   Prompt: What features should the front-end have?
   Response: A clean, futuristic design with a ticker at the top.
   ```
