## Code Examples
Here are some example prompts and code snippets to help you create AI-animated African folktales.

### Story Generation
```plaintext
"Write me a short story of two glamorous Nigerian ladies with taste of class who s for money in wrong places and ended up in madness in a pizza style with narrator's voice."
```

### Character Descriptions
```plaintext
"Write the main character's description."
```

### Visual Creation
```plaintext
"Write text to image prompt of each paragraph of the story in pizza style."
```

### Video Editing
When using CapCut, follow these steps:
1. Import all the images and videos.
2. Arrange them in the timeline according to the story.
3. Add the voiceover and align it with the visuals.
4. Add animations to the visuals.
5. Review and export the final video.

### Voiceover
Use your own voice for the voiceover. Record the voiceover and import it into CapCut. Align the voiceover with the visuals to create a cohesive narrative.

### Animations
In CapCut, select the visuals and add animations. Use the combo animation feature to make the visuals more engaging.

By using these code examples and prompts, you can streamline the process of creating AI-animated African folktales.
