## Practical Guide
This guide provides detailed steps and best practices for creating AI-animated African folktales.

### Step 1: Generate the Story Script
Start by using an AI tool like ChatGPT to generate the story script. Provide a detailed prompt to guide the AI. Example prompt: `"Write me a short story of two glamorous Nigerian ladies with taste of class who s for money in wrong places and ended up in madness in a pizza style with narrator's voice."`

### Step 2: Refine the Script
Copy the generated script into an editing app like Microsoft Word or WordPad. Refine the script to remove robotic language and add cultural elements. This step ensures the story feels authentic and engaging.

### Step 3: Create Character Descriptions
Generate detailed descriptions of the main characters. Example prompt: `"Write the main character's description."` This helps in creating consistent visuals for the characters.

### Step 4: Generate Visuals
Use text-to-image AI tools like Whisk AI to create visuals based on the story paragraphs. Example prompt: `"Write text to image prompt of each paragraph of the story in pizza style."` This ensures the visuals align with the narrative.

### Step 5: Assemble the Video
Use video editing software like CapCut to assemble the visuals and voiceover. Ensure the visuals align with the voiceover to create a cohesive narrative.

### Step 6: Add Animations
Animate the visuals to make the video more engaging. Use tools within CapCut to add animations. This step enhances the visual appeal of the video.

### Step 7: Finalize and Export
Review the video, make necessary adjustments, and export the final product. Ensure the video meets quality standards and effectively tells the story.

### Best Practices
- **Refine the Script**: Always refine the AI-generated script to add cultural elements and remove robotic language.
- **Consistent Characters**: Use reference images to maintain consistent character appearances throughout the video.
- **Natural Voiceover**: Use your own voice for the voiceover to avoid robotic content and potential issues with platforms like YouTube.

### Common Pitfalls
- **Robotic Scripts**: Failing to refine the AI-generated script can result in robotic and unnatural storytelling.
- **Inconsistent Characters**: Not using reference images can lead to inconsistent character appearances.
- **Overuse of AI Voiceovers**: Using AI voiceovers can make the content feel robotic and may lead to issues with content platforms.

By following this practical guide, you can create AI-animated African folktales that are engaging, culturally relevant, and visually appealing.
