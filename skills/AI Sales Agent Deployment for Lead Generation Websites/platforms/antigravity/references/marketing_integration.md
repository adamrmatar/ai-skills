# Marketing Integration for AI Sales Agents

## Ad Campaign Creation
1. **Asset Generation**:
   - Use platform tools to create Instagram/Meta ads
   - Ensure they match the established brand aesthetic
2. **Campaign Strategy**:
   - Use AI to build campaign structure:
     - Target radius
     - Budget range
     - Conversion goal (test drives for dealers, viewings for realtors)
   - Get audience segment recommendations
   - Get ad set structure
   - Get copy variants for each creative

## Campaign Execution
- Start with small tests ($20/day for 3 days)
- For car dealers:
  - Objective: Lead generation
  - Conversion event: Booking confirmation
- For realtors:
  - Objective: Form submissions
  - Longer warm-up period (1 week)
  - More patience with results

## Optimization
- Let data determine which creatives perform
- Don't judge by impressions - focus on actual conversions
- For dealers: booked test drives
- For realtors: viewing appointments

## Client Reporting
- Use white-labeled dashboard
- Show:
  - Number of inquiries handled
  - Conversion rates
  - Booking volume
  - Ad performance
- Position as "your system" not third-party tools
