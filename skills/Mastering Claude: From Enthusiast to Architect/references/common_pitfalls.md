# Common Pitfalls
This document highlights common mistakes and how to avoid them when using Claude AI.

### Overloading Context
Claude's context window can become sloppy past 50% capacity. Use `/compact` to summarize older history and `/context` to monitor token usage.

### Manual Management
Avoid becoming a bottleneck by automating repetitive tasks. Use skills and scheduled tasks to streamline workflows.

### Untrusted Sources
Be cautious when downloading skills from unverified sources. Stick to official Anthropic skills or trusted community marketplaces.

### Lack of Trust
Transitioning to autonomous systems requires trust. Start with low-stakes routines and gradually scale to more critical tasks.

For a detailed workflow guide, refer to the Practical Guide.
