# Code Examples
This document provides concrete examples of prompts, skills, and workflows for Claude AI.

### Example Skill: Weekly Client Reporting
```markdown
1. Pull data from Google Drive.
2. Summarize key metrics.
3. Generate a PowerPoint deck.
4. Send the report via email.
```

### Example Prompt: File Organization
```markdown
Organize the downloads folder by file type, rename files consistently, and generate a summary of contents.
```

### Example Workflow: Parallel Testing
```markdown
1. Spin up a work tree for feature implementation.
2. Spin up a second work tree for bug fixing.
3. Spin up a third work tree for writing tests.
4. Run all workflows in parallel.
```

For more on best practices and pitfalls, refer to the Common Pitfalls document.
