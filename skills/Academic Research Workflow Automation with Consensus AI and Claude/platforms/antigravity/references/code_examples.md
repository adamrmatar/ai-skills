# Code Examples

## Introduction
This document provides examples of code snippets and prompt templates for integrating Consensus AI with Claude and executing academic research workflows.

## Code Snippets

### Connecting Consensus AI to Claude
```markdown
**Prompt Template for Connecting Consensus AI:**
1. Open Claude and navigate to the customization section.
2. Click on 'Browse Connectors' and search for Consensus AI.
3. Click on Consensus AI and follow the connection process, including signing in.
```

### Uploading Pre-Built Skills
```markdown
**Prompt Template for Uploading Skills:**
1. Download the desired skill (e.g., literature review helper) from the Consensus AI website.
2. In Claude, go to the skills section and upload the downloaded skill.
```

### Creating Custom Skills
```markdown
**Prompt Template for Creating Skills:**
1. Use Claude's skill creator to develop custom skills tailored to your specific research needs.
2. Follow the prompts to define the processes and outcomes you want the skill to achieve.
```

### Executing Research Tasks
```markdown
**Prompt Template for Literature Review:**
Find research gaps for my PhD project on [Your Topic].
How thorough should the literature search be? [Specify, e.g., 'Standard 10 searches per idea'].
Are there any subareas you want to adjust before I proceed? [Specify, e.g., 'No, this looks great. Go ahead with it.']
```

## Conclusion
These code examples and prompt templates will help you effectively integrate Consensus AI with Claude, automating and enhancing your academic research workflows.
