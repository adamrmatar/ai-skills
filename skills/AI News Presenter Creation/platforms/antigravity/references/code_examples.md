# Code Examples and Prompt Templates

Here are concrete examples and templates for creating AI news presenters.

## ChatGPT Prompt for Script Translation
```
I have two newscasts. One is presented in English and the other is presenting in French. Give me their intro in news in breaking news. Also, translate this [English news segment] to French for the second presenter.
```

## Grok AI Prompt for Presenter Image Generation
```
Change her outfit to newscaster. Add a male presenter, a foreigner. Place them in a modern newsroom facing camera. Add a breaking news banner behind them. At the top right, add 'Live' in a red square.
```

## Tone Specification for AI Presenters
```
The lady presenter is to present in a confident, authoritative, and composed tone with calm urgency suited for delivering serious national news. The male presenter should have a smooth, formal, and refined tone with a steady, elegant flow suited for international broadcast style.
```

## CapCut Editing Steps
1. Import generated videos and audio.
2. Add background music by searching for 'news' in the audio library.
3. Adjust volumes to ensure clarity.
4. Insert relevant images and apply animations (e.g., fade-in, fly-out).
5. Export the final video after previewing.

These examples provide a clear starting point for each stage of the creation process.
