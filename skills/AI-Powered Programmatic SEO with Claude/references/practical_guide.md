# Practical Guide

## Identifying Topic Patterns
To identify topic patterns, use the following prompt:
```plaintext
Prompt: Generate topic patterns for [your niche] with the goal of promoting [your product or goal].
```
This will give you a list of repeatable keyword structures.

## Generating Keywords
Once you have a pattern, generate keywords using:
```plaintext
Prompt: Generate 20 keywords based on the pattern [selected pattern].
```
Validate these keywords using a bulk keyword tool to ensure they have some search volume.

## Creating Page Templates
Develop a page template with Claude using:
```plaintext
Prompt: Create a page template for the pattern [selected pattern] that includes a unique angle, fresh data, SEO and AIO optimization, and goal blending.
```
Refine the template iteratively until satisfied.

## Creating Claude Skills
Turn your pattern and template into a reusable Claude skill using:
```plaintext
Prompt: Create a Claude skill for the pattern [selected pattern] and template [generated template].
```
Test the skill by generating a new page template.

## Scaling Content Generation
Use Claude commands or spawn multiple sub-agents to generate multiple pages simultaneously. For example:
```plaintext
Prompt: Generate a new page using the skill [skill name] with the keyword [new keyword].
```
