# Code Examples and Prompt Templates

This document provides examples of prompts and code snippets for automating tasks with Genspark Claw.

## File Organization
```plaintext
Organize files on my desktop into folders based on content.
```

## Data Processing
```plaintext
Take raw sales data, create a summary, build a chart, format everything, and send it out.
```

## Email Automation
```plaintext
Review my calendar, check my email, and pull in any important news about the cookie industry.
```

## Scheduling Recurring Tasks
```plaintext
Run this task every morning at 8am.
```

## Meeting Summaries
```plaintext
Record a meeting, generate a transcript, and summarize the key points and action items.
```

For best practices and common pitfalls, refer to [Common Pitfalls](references/common_pitfalls.md).
