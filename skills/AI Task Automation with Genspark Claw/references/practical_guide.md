# Practical Guide to Using Genspark Claw

This guide provides step-by-step instructions on how to use Genspark Claw to automate tasks on your computer.

## Installation
1. **Download the Desktop App**: Visit the Genspark website and download the desktop app.
2. **Set Up Cloud Version**: For longer-running tasks, set up the cloud version by selecting the cloud option during installation.

## Task Automation
1. **Define Your Task**: Use the input box to describe the task you want Claw to complete. Example: `Organize files on my desktop into folders based on content.`
2. **Specify the Folder**: Click on the folder icon to direct Claw to the specific area on your computer.
3. **Run the Task**: Click the run button to execute the task. Claw will process the files and organize them accordingly.
4. **Review Results**: Check the summary provided by Claw and verify the organization on your desktop.

## Scheduling Recurring Tasks
1. **Define the Task**: Use the input box to describe the task you want to schedule. Example: `Run this task every morning at 8am.`
2. **Set the Schedule**: Use the scheduling feature to set the task to run at the specified time.
3. **Switch to Cloud Computing**: For longer-running tasks, switch to the cloud version to ensure they run even when your computer is off.
