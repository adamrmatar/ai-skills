# Common Pitfalls and Best Practices

## Process Interviewer
**Pitfall**: Jumping into action without full context.
**Best Practice**: Ensure a thorough understanding of the task by answering all questions posed by the Process Interviewer.

## Prompt Master
**Pitfall**: Using unstructured, error-prone prompts.
**Best Practice**: Always optimize prompts using the Prompt Master skill for complex tasks.

## Humanizer
**Pitfall**: AI-generated text sounding robotic.
**Best Practice**: Run the Humanizer skill on any AI-generated text intended for public consumption.

## Fact Checker
**Pitfall**: Factual inaccuracies in AI-generated content.
**Best Practice**: Verify all factual claims using the Fact Checker skill.

## Find Skills
**Pitfall**: Missing out on useful skills.
**Best Practice**: Regularly use the Find Skills skill to discover and install new skills.

## Front-End Slides
**Pitfall**: Low-quality presentations.
**Best Practice**: Specify brand guidelines to ensure high-quality, consistent presentations.

## Decision Toolkit
**Pitfall**: Making uninformed decisions.
**Best Practice**: Use the Decision Toolkit skill to systematically analyze decisions.

## MCP Builder
**Pitfall**: Lack of software integration.
**Best Practice**: Use the MCP Builder skill to create custom connectors for seamless software integration.
