# Code Examples

## Payroll AI Agent Setup
```markdown
1. Navigate to QuickBooks homepage.
2. Click on Feed in the left sidebar.
3. Click on 'See how it works'.
4. Set cadence and choose days of the week.
5. Ensure employee consent for text communications.
```

## Job Posting Creation
```markdown
1. Navigate to Teams app > Recruiting.
2. Fill in job details.
3. Generate job description if needed.
4. Set up application form.
5. Publish job.
```

## Onboarding New Employees
```markdown
1. Go to all apps, team, and then employees.
2. Click on Add Employee and enter the employee's details.
3. Follow the onboarding steps, including document signing and HR workflows.
```

## Setting Up HR Workflows
```markdown
1. Navigate to the team app and click on HR Workflows.
2. Add a template from the library or create a new one.
3. Assign steps to specific team members and customize as needed.
4. Start the workflow for the relevant employee.
```

For best practices and common pitfalls, refer to [Common Pitfalls](references/common_pitfalls.md).
