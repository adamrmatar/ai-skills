# Optimizing Ai Agents To Reduce Hallucinations And Improve Efficiency

This skill teaches AI agents how to optimize their performance by reducing hallucinations, minimizing token waste, and improving accuracy through five key techniques: semantic tool selection, GraphRAG, multi-agent validation, neuro-symbolic guardians, and runtime guardians.

## Setup & Installation

This skill is agent-agnostic and packaged for multiple developer agents. Find your target platform below:

### 💻 Antigravity
_Copy the contents of `platforms/antigravity/` to `~/.gemini/config/skills/Optimizing AI Agents to Reduce Hallucinations and Improve Efficiency/`._

### 💻 Cursor
_Copy `platforms/cursor/Optimizing AI Agents to Reduce Hallucinations and Improve Efficiency.mdc` to your project's `.cursor/rules/` directory._

### 💻 Claude
_Append the contents of `platforms/claude/CLAUDE.md` to the target project's `CLAUDE.md` file._

### 💻 Copilot
_Copy `platforms/copilot/Optimizing AI Agents to Reduce Hallucinations and Improve Efficiency.instructions.md` to your project's `.github/instructions/` directory._

### 💻 Windsurf
_Append the contents of `platforms/windsurf/.windsurfrules` to the target project's `.windsurfrules` file._
