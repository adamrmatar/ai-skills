# Structured Specification Prompting For Autonomous Ai Agents

Learn how to create structured specifications that enable AI agents to autonomously complete complex tasks like generating PowerPoint decks with high accuracy and efficiency.

## Setup & Installation

This skill is agent-agnostic and packaged for multiple developer agents. Find your target platform below:

### 💻 Antigravity
_Copy the contents of `platforms/antigravity/` to `~/.gemini/config/skills/Structured Specification Prompting for Autonomous AI Agents/`._

### 💻 Cursor
_Copy `platforms/cursor/Structured Specification Prompting for Autonomous AI Agents.mdc` to your project's `.cursor/rules/` directory._

### 💻 Claude
_Append the contents of `platforms/claude/CLAUDE.md` to the target project's `CLAUDE.md` file._

### 💻 Copilot
_Copy `platforms/copilot/Structured Specification Prompting for Autonomous AI Agents.instructions.md` to your project's `.github/instructions/` directory._

### 💻 Windsurf
_Append the contents of `platforms/windsurf/.windsurfrules` to the target project's `.windsurfrules` file._
