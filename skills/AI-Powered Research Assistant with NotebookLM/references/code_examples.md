# Code Examples
Here are some example prompts you can use with NotebookLM.

### Summarization
```markdown
Summarize the key points from the provided sources about the cookie launch strategy.
```

### Risk Analysis
```markdown
What are the biggest risks mentioned across all the different sources?
```

### Mind Map Generation
```markdown
Generate a mind map showing the trends in the cookie industry.
```

### Audio Overview
```markdown
Generate an audio overview focused on the cookie launch plan.
```

### Custom Notes
```markdown
Add a custom note with the following feedback: 'The customer really liked the new cookie.'
```

### Sharing
```markdown
Share the notebook with the following people: [email1@example.com, email2@example.com].
```
