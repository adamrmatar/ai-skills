### Code Examples and Prompt Templates

This document provides example prompts and code snippets to help you get started with Claude Co-Work.

#### Organizing Scanned Documents
Use the following prompt to organize scanned documents:
```markdown
"Claude, please organize the scanned documents in the 'Scan Inbox' folder. Rename each file based on its content, including the date, company name, and document type. Move them to the appropriate year folder."
```

#### Creating Skills
Use the following prompt to create a skill:
```markdown
"Claude, create a skill that automatically organizes scanned documents into predefined folders based on their content. Save this skill as 'Document Organizer'."
```

#### Automating Report Generation
Use the following prompt to automate report generation:
```markdown
"Claude, generate a monthly report summarizing the sales data from the 'Sales Data' folder. Include charts and key metrics."
```

#### Interacting with Local Files
Use the following prompt to interact with local files:
```markdown
"Claude, please analyze the data in the 'Sales Data' folder and provide a summary of key metrics."
```

#### Context-Aware Assistance
Use the following prompt to get context-aware assistance:
```markdown
"Claude, based on the current project status, what are the next steps I should take?"
```

These examples demonstrate how to use Claude Co-Work to automate tasks, organize files, and provide context-aware assistance. By leveraging these prompts, you can enhance your productivity and streamline your workflows.
