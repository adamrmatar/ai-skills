### Practical Guide
**Setting Up Custom Instructions**:
1. Navigate to the custom instructions settings.
2. Write clear and specific instructions. Example:
   - Default to direct, specific answers.
   - Skip the hedging and the four-option lists.
   - When you give me a number, tell me where it came from.
   - Push back if you think I'm wrong instead of agreeing by default.

**Creating Projects**:
1. Create a new project.
2. Upload relevant documents to the project's knowledge base.
3. Define project-specific instructions.

**Enabling Memory**:
1. Go to settings and turn on memory.
2. Regularly review and delete stale memory entries.

**Using Adaptive Thinking**:
1. Select the effort level (low, medium, high, max) based on the task complexity.

**Generating Artifacts**:
1. Enable code execution and file creation in settings.
2. Generate artifacts such as documents, spreadsheets, and interactive tools.
3. Store and share artifacts from the dedicated artifacts tab.

Following these steps will help you leverage Claude AI's advanced features effectively.
