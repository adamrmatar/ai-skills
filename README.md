# My AI Agent Skills

A collection of custom AI agent skills generated automatically from curated videos on [25experts.com](https://25experts.com).

These skills are compiled for multiple developer agents: Antigravity, Cursor, Claude Code, GitHub Copilot, and Windsurf.

## 🛠️ Available Skills

- **[AI Agent Operating System Management](./skills/AI Agent Operating System Management/README.md)**: A comprehensive skill for managing AI agents through an operating system that handles scheduling, memory, tools, identity, observability, and guardrails to ensure reliable and efficient operation.
- **[Agentic Experimentation for High-Stakes ML Systems](./skills/Agentic Experimentation for High-Stakes ML Systems/README.md)**: A skill for AI agents to safely and efficiently experiment with high-stakes ML systems by modifying features, training models, and deploying pipelines in isolated environments while maintaining production integrity.
- **[Automate Tasks with Genspark Claw](./skills/Automate Tasks with Genspark Claw/README.md)**: Learn how to automate tasks using Genspark Claw, an AI-powered tool that interacts with your computer to complete tasks such as file organization, data analysis, email automation, and meeting summaries.
- **[Automating Email Responses and Meeting Scheduling with OpenClaw](./skills/Automating Email Responses and Meeting Scheduling with OpenClaw/README.md)**: This skill enables an AI agent to automate email responses and meeting scheduling using OpenClaw, a digital assistant platform. The agent will learn to set up OpenClaw on a VPS, configure integrations with Gmail, Google Drive, and Google Calendar, and execute workflows for responding to sponsorship emails and scheduling meetings.
- **[Claude Cowork Workflows](./skills/Claude Cowork Workflows/README.md)**: A comprehensive skill for setting up and optimizing Claude Cowork workflows, including file access, projects, connectors, skills development, and testing methodologies.
- **[Claude Design Automation](./skills/Claude Design Automation/README.md)**: A systematic approach to generating brand-aligned design assets (presentations, social media images, websites) in Claude Design with minimal iterations by pre-establishing design systems, templates, and using skills for copy generation.
- **[Handoff Skill](./skills/Handoff Skill/README.md)**: Compact the current conversation into a handoff document for another agent to pick up, summarizing the conversation and suggesting next steps.
- **[Improve Codebase Architecture](./skills/Improve Codebase Architecture/README.md)**: A systematic approach to refactoring and improving codebase architecture by identifying and deepening shallow modules, improving locality and leverage, and establishing clear seams and interfaces.
- **[RAG vs Fine-Tuning Decision Framework](./skills/RAG vs Fine-Tuning Decision Framework/README.md)**: A comprehensive skill for determining when to use Retrieval-Augmented Generation (RAG) versus fine-tuning in generative AI systems, based on real-world production scenarios.
- **[Vector Database Integration for Generative AI Systems](./skills/Vector Database Integration for Generative AI Systems/README.md)**: Learn how to integrate vector databases into generative AI systems for efficient semantic similarity search and context retrieval.
- **[claude-cowork-workflows](./skills/claude-cowork-workflows/README.md)**: Best practices for co-working with Claude (Desktop, Projects, Artifacts) to accelerate programming, code editing, and system design.
- **[improve-codebase-architecture](./skills/improve-codebase-architecture/README.md)**: How to refactor, format, and structure a codebase that has been cluttered or corrupted by AI-generated boilerplate code (slop).
